# Mogz Visuals

Mogz Visuals is a photography and creative studio in Juba. This is the official website and portfolio.

## Features

- **Immersive Portfolio:** A visually-driven portfolio experience with smooth scrolling and animations.
- **Private Client Galleries:** Secure, password-protected galleries for clients to view and download their collections.
- **Direct Link Access:** Clients can receive a direct link to their private collection, bypassing the need to enter credentials every time.
- **One-Click Downloads:** Download an entire collection as a single zip file.
- **Responsive Design:** Fully responsive and optimized for all devices.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **CMS:** [Sanity](https://www.sanity.io/)
- **Deployment:** [Vercel](https://vercel.com/)
